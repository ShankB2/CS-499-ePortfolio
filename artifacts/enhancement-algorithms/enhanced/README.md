# Enhancement 2: Algorithms and Data Structures (CS-370)

## Overview
This enhancement focuses on improving the Pirate Intelligent Agent project originally developed in CS-370. The project demonstrates reinforcement learning concepts using deep Q-learning and neural networks to train an intelligent pirate agent to solve a maze environment.

The enhanced version improves project organization, evaluation, maintainability, performance tracking, and reinforcement learning validation.
