# Original Pirate Intelligent Agent Artifact (CS-370)

## Overview
This folder contains the original Pirate Intelligent Agent artifact developed during CS-370 coursework before enhancements were implemented.

The original project demonstrated reinforcement learning, neural networks, replay memory, and maze navigation using deep Q-learning.

- Original reinforcement learning implementation

- Original project documentation
