# Software Design Enhancement (CS-499)

## Overview
This project enhances a Contact Service application originally developed in CS-320. The enhancement focuses on improving software design, validation, security, and maintainability.

## Structure
- **original/** – Contains the original unmodified artifact
- **enhanced/** – Contains the improved version with design enhancements
- **screenshots/** – Evidence of testing and successful execution
- **code-review/** – Code review script and analysis

## Enhancements Implemented
- Input validation for all fields
- Prevention of duplicate contact IDs
- Improved error handling using exceptions
- Separation of concerns between data and service layers
- Use of HashMap for efficient data storage
- Immutable contact ID for data integrity
- Read-only access to stored data
- Comprehensive unit testing with JUnit
